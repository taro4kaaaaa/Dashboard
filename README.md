## ✨ Features

- 📋 Создание и удаление задач
- 🔄 Изменение статуса (todo / in-progress / done)
- 🎯 Приоритет задач (low / medium / high)
- 🔍 Фильтрация задач
- 📄 Пагинация
- 📊 Аналитика с графиками (Chart.js)
- 💾 Persist (сохранение в localStorage)

---

## 🛠 Tech Stack

- React
- TypeScript
- Zustand (state management)
- React Router
- Chart.js
- CSS
